# Weather APP: Daily Temperature Guide 

![This is a alt text.](/public/weather-app-image.png "This is a sample image.")

Weather App is an easy-to-use, daily temperature guide that provides accurate weather information for any location. This app is built with React and displays current weather data, including temperature, conditions, and 10-day forecast. Stay updated with the latest weather updates with this app.


## Tech Stack
* Built with React JS for creating UI components
* Implements Redux for efficient state management
* Utilizes Redux Toolkit for optimized and simplified Redux code
* Makes API calls with Axios for retrieving weather data
* Styled using Tailwind CSS for a sleek and modern look
* Provides accurate and reliable weather information with a smooth user experience.
